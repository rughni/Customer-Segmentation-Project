# Customer Segmentation Project

## Overview

This project uses K-Means Clustering to segment customers based on their annual income and spending behavior.

## Objective

* Analyze customer behavior
* Group customers into meaningful segments
* Generate business insights

## Tools Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Dataset Features

* CustomerID
* Age
* Gender
* AnnualIncome
* SpendingScore
* PurchaseFrequency

## Methodology

1. Data Collection
2. Data Preprocessing
3. Elbow Method
4. K-Means Clustering
5. Customer Segmentation
6. Data Visualization

## Results

The Elbow Method identified 5 customer clusters based on income and spending patterns.

## Business Insights

* Premium customers can be targeted with exclusive offers.
* Medium-income customers can be encouraged through loyalty programs.
* Budget-conscious customers respond well to discounts.

## Conclusion

Customer segmentation helps businesses understand customer behavior and create targeted marketing strategies for better decision-making.
